How to use the API

# POST request to generate a QR code
curl -X POST https://your-domain.com/api/v1/generate \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -d '{"url": "https://example.com"}'

# Response
{
  "status": "success",
  "image_url": "https://...",
  "target_url": "https://example.com"
